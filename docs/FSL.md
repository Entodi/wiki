FSL is a comprehensive library of analysis tools for FMRI, MRI and DTI
brain imaging data.

To use FSL GUI, select FSL from the TReNDS interactive apps on [https://hemera.rs.gsu.edu/](https://hemera.rs.gsu.edu/).
